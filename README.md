This portfolio is engineered for a "Very, Very Fast" user experience:
- **FCP (First Contentful Paint):** ~144ms.
- **Extreme Image Optimization:** All assets converted to compressed JPEG (~80% quality), reducing payload by >80%.
- **Zero Layout Shift:** Implemented fixed aspect-ratios and dimensions for all media.
- **GPU Acceleration:** High-fidelity animations utilize `will-change` property for buttery smooth 60fps interaction.
- **Modern Delivery:** Resource hints (`preconnect`, `dns-prefetch`), asynchronous image decoding, and deferred JavaScript execution.
